# changingchoices

Changing Choices? Primary and Secondary Effects Through Times of Educational Contraction 

The author cannot share the data underlying this article publicly due to ethical and privacy requirements of the data provider, Statistics Finland. An anonymized code that identifies the procedures followed to produce the dataset analyzed and to replicate the analysis in the article is provided in this repository. 
 
Most of the raw data used is updated continuously.  
Data version used for analysis: 1/7/2024  
 
Programme used for the analysis: Stata 18 
changing choices_data and variables.do file consists of the code and comments for constructing the dataset used in the manuscript.  
 
changing choices_analysis.do file consists of the code and comments to replicate the analysis in the manuscript.
